import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs groupId="languages">
<TabItem value="py" label="Python">

Install the latest, [Python client `v4`](/developers/weaviate/client-libraries/python), by adding `weaviate-client` to your Python environment with `pip`:<br/><br/>


```bash
pip install -U weaviate-client
```

</TabItem>
<TabItem value="js" label="JS/TS">

Install the latest, [JS/TS client `v3`](/developers/weaviate/client-libraries/typescript/typescript-v3), by adding `weaviate-client` to your project with `npm`:<br/><br/>

```bash
npm install weaviate-client
```

</TabItem>
<TabItem value="go" label="Go">

Add `weaviate-go-client` to your project with `go get`:<br/><br/>

```bash
go get github.com/weaviate/weaviate-go-client/v4
```

</TabItem>
<TabItem value="java" label="Java">

Add this dependency to your project:<br/><br/>

```xml
<dependency>
  <groupId>io.weaviate</groupId>
  <artifactId>client</artifactId>
  <version>4.8.3</version>  <!-- Check latest version: https://github.com/weaviate/java-client -->
</dependency>
```

</TabItem>
</Tabs>
