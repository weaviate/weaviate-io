When connecting to a Weaviate Cloud cluster, you need an API key and the REST endpoint URL for authentication.
To retrieve your connection details, follow these steps:

:::note REST Endpoint vs gRPC Endpoint
When using an official Weaviate [client library](/developers/weaviate/client-libraries), you need to authenticate using the `REST Endpoint`. The client will infer the gRPC endpoint automatically and use the more performant gRPC protocol when available.
