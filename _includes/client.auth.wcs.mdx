:::tip WCS + Weaviate client
Each Weaviate instance in Weaviate Cloud Services (WCS) is pre-configured to act as a token issuer for OIDC authentication. 
:::

The "Resource Owner Password Flow" method the easiest way to be authenticated by a WCS instance. 

You can do this by passing on your WCS `username` and `password` to the client as per the pattern below. You do not need to set the optional `scope` parameter.
