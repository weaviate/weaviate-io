## Overview

Each Weaviate object belongs to one (and only one) class. This page covers the essential topics in relation to managing Weaviate classes.

## Create a class

You can add a class to the Weaviate [schema](../concepts/data.md#class-collections) manually, or have Weaviate create it during object creation/import (auto-import).

### Auto-schema

To use auto-schema, all you need to do is to create/import objects.

If you do so to a class that does not exist, Weaviate will create the class by inferring properties as required.

:::info Read more about Auto-schema
See [this page](../config-refs/schema.md#auto-schema)
:::

### Manual creation

To create a class, you need a class name at a minimum with the `class` parameter.
