## Generate deterministic IDs

You can generate an ID based on your data object. 

:::info
Gerenating IDs from an objects is not random, as the same value will always the same ID.<br/>
Using deterministic IDs allows us to avoid inserting duplicate objects, as Weaviate throws an error if you provide a duplicate ID. 
:::
