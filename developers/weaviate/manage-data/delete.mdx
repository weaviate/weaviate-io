## Overview

This page will show you how to delete individual objects in Weaviate.


## Requirements

To delete an object, you'll need its [id](../api/graphql/additional-properties.md#id) and class name.

To delete multiple objects, you'll need their class name, and a filter that matches them.

## Batch delete multiple objects

To delete multiple objects at once, make a [`DELETE` request to the `/v1/batch/objects` REST API endpoint](../api/rest/batch.md#batch-delete), or use the client code below. Specify which objects to delete using the same kind of [`where` filter](../search/similarity.md) used in similarity search.

For limitations, optional parameters, and the response format, see the [Batch delete API reference](../api/rest/batch.md#batch-delete).
