## Overview

Learn how to perform semantic searches using Weaviate.

## Semantic search methods

### Using an input text

`Get` + `nearText`

### Using an object

`Get` + `nearObject`

### Using a vector

`Get` + `nearVector`

## Limit the returned results

### `limit` argument

`Get` + `near<>` + `limit`
`Get` + `near<>` + `limit` + `offset`

### Distance parameters (`distance` and `certainty`)

`Get` + `near<>` + `distance`
`Get` + `near<>` + `certainty`

Reference to distance metrics

### groupBy

`Get` + `near<>` + `limit` + `groupBy`
