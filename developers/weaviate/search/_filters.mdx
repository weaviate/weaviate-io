## Overview

Learn how to add boolean (`where`) filters to your search queries.

### Single filter examples

`Get` + `where`
`Get` + `near<>` + `where`

### Multiple filters

`Get` + `where` (`and`)
`Get` + `where` (`or`)
`Get` + `where` (nested operands?)
`Get` + `near<>` + `where` (`and`)
