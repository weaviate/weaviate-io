## <i class="fa-solid fa-code"></i> Installation

The latest Weaviate Python client library can be installed using pip. The client library is tested on Python 3.8 and later. Install it using the following command:

```bash
pip install -U weaviate-client
```

The latest major version is `v4` (e.g. `4.4.4`). You can check the version like so:

```bash
pip show weaviate-client
```

## <i class="fa-solid fa-code"></i> Basic usage

From Python, you can load the Weaviate client library like so:

```python
import weaviate
```

The client provides sets of helper classes to make it easier to interact with Weaviate. They are organized into submodules, such as `weaviate.config`, `weaviate.batch`, `weaviate.query` and so on.
