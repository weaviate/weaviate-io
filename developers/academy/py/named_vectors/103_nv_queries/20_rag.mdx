Retrieval augmented generation (RAG) queries can also benefit from use of named vectors. The benefit comes from an improved retrieval of data, which can in turn improve the quality of the generation.

## <i class="fa-solid fa-chalkboard"></i> Single prompt queries

### <i class="fa-solid fa-code"></i> Code

This example finds entries in "MovieNVDemo" based on their similarity to [this image of the International Space Station](https://upload.wikimedia.org/wikipedia/commons/thumb/0/04/International_Space_Station_after_undocking_of_STS-132.jpg/440px-International_Space_Station_after_undocking_of_STS-132.jpg). Then, instructs the large language model to translate the title of each movie into French.

Each of the results are then printed out to the console.
